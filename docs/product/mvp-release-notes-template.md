# OpenPrecedent 0.1.0 MVP Release Notes Template

Use this template when publishing the `v0.1.0` MVP release on GitHub.

## Summary

OpenPrecedent `0.1.0` is the first published MVP release: a local-first, research-oriented, developer-facing baseline for decision replay and precedent retrieval.

## Included In This Release

- stable Rust `openprecedent` CLI
- local SQLite-backed case, event, decision, replay, and precedent workflows
- OpenClaw import and collection support
- fixture and collected-session evaluation paths
- release validation and quickstart documentation

## Strongest Validated Usage Mode

- local single-agent workflow
- OpenClaw as the strongest integrated runtime
- source-build installation path
