# Competitive Landscape

Date: 2026-03-09

## Summary

The market is active, but split across layers.

The closest long-term competitor identified so far is ElixirData, which combines context, control, and decision traces. Nearby categories include:

- observability and tracing
- memory and context graph systems
- governance and control platforms

## Category View

### Observability / Tracing

- LangSmith
- Langfuse
- Braintrust
- Arize Phoenix
- OpenLIT
- Microsoft Foundry
- AWS AgentCore
- OpenAI Agents SDK tracing

These products are strong at answering:

- what the agent did

But they are not centered on reusable decision objects.

### Memory / Context

- Zep
- Letta
- Mem0
- Supermemory
- MemLayer
- Ryumem
- Recallr
- Mnemosyne

These products are strong at answering:

- what the agent should remember

But they are not centered on audit-ready decision replay.

### Decision / Governance

- ElixirData
- IBM watsonx Orchestrate
- AI Agentree

This is the closest category to OpenPrecedent's long-term direction.

## Key Market Takeaways

- tracing is not a white space
- memory is not a white space
- governance is not a white space
- the main opening is the combination layer centered on decision objects

## Open Source Implications

Open-source pressure is strongest in observability and memory:

- Langfuse, Phoenix, and OpenLIT are meaningful open-source observability options
- Zep's Graphiti, Letta, Mem0, and Supermemory all show open-source or open-core pressure on the memory layer

The decision layer is still much less occupied by strong open-source products. That is an important opening for OpenPrecedent.
